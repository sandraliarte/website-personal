About Spotless Theme

Spotless - a carefully crafted theme that lets you display your work the clean, fancy way. Some of the theme's strong points are the awesomne iGallery, the sortable masonry gallery, the multipurpose slider. The responsive behaviour allows the content to easily adjust for any screen resolution in order to produce the optimal user experience for different use cases and enviroments, all built into one website.

HTML structure

There are 9 sections types available for this template (home, about, iGallery, portfolio, single project, blog, single blog post, videos and contact), you can add/remove as many sections as you want. Please note that all pages must be stored under the 'pages/' folder.

IMPORTANT! - please read this
You can change most of the template's look and feel by editing css/main.css near (textColor01 to textColor03 and backgroundColor01 to backgroundColor04).
Other settings/colors can be changed from css/sections.css.

IMPORTANT! - please read this
You can an unlimited number of sections that are not present within the main website menu.

Example: You can add unlimited number of single project pages that are being accessed from the portfolio menu, same could happen with videos pages.

IMPORTANT! - please read this
The theme is Ajax powered, meaning its pages are being dynamically loaded making a page available as soon as it has been loaded. Because of the this it needs to run on a web server in order to work. Please upload it on your server and edit it there. Alternatively you can install a web server on your local computer, please see Wamp (windows) or Mamp (Mac OS).

CSS files

The template's custom css files are located under the css folder:

1. main.css - responsable with general colors/look and feel
2. sections.css - responsable with sections styles
3. elastislide.css - responsable with styles for the horizontal carousel
4. eislideshow.css - responsable with styles for the single project slider
5. reset.css (resets the browser)

Additional CSS files belong to Skeleton Framework (skeleton/) and to the Degrade Slider (degrade_slider/css).

Template's sound

Spotless supports ambient sound/track that will loop continously. The template's audio files are located within the "sounds/" folder.
You can remove this feature by editing "SpotlessWebsite.js"
and change te value of isWebsiteSound to false.
You can change the volume by editing "soundVolume"
You can change the track by editing "soundResource"

IMPORTANT! - please read this
Some of the browsers can play MP3 sond files, others can play OGG files, so you need to provide the same sound within two formats (.mp3 andd .ogg) - see sounds folder.

Degrade Slider

Degrade slider is a multipurpose, fully customizable slider, with responsive behavior and all-in-one features. It sports a 40+ options variation, the slider‘s appearance can be customized through its‘ 8 different shadows, 4 different corner style, 6 different control types, unlimited colors, shortcodes, captions style, fullscreen behavior, giving you the possibility of making literally hundreds of different sliders.

The slider can be also used as a banner, can play video, can display captions wherever you want, can be used as advertising tool as well as display panel.
You can find additional documentation about the slider within the downloaded files - silder_docs.
